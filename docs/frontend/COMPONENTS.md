# רכיבי UI

## מיקום

`client/src/components/` — רכיבים משותפים לפי תחום (`ui/`, `game/`, `statistics/`, `admin/`, וכו').

## דוגמאות קבצים (לתיעוד)

| קובץ | תיאור קצר |
|------|------------|
| `Button.tsx` | כפתורים |
| `Card.tsx` | כרטיסים |
| `Navigation.tsx` | ניווט ראשי |
| `GameSessionHud.tsx` | HUD במשחק |
| `TrendChart.tsx` | גרף מגמה |
| `PieChart.tsx` | גרף עוגה |
| `LeaderboardTable.tsx` | טבלת דירוג |
| `ErrorBoundary.tsx` | תפיסת שגיאות |

רשימה מלאה: עיין בקבצי `components/**/*.tsx` בפרויקט.
